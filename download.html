Download

You can download this project in either zip or tar formats.

Get the source code on GitHub: inaka/edis

You can also clone the project with Git by running: 

$ git clone git://github.com/inaka/edis

Installation

You should install erlang and rebar first.

Then clone the project and compile it:

$ git clone git://github.com/inaka/edis
$ cd edis/
edis$ make

If everything went smoothly, you should be able to run your edis instance:

edis$ bin/edis

You can also install it on your system:

edis$ sudo make install

Then you will be able to run edis everywhere:

~$ edis

You can even install it as a deamon:

edis$ sudo make service

Edis accepts one parameter on command-line: the config file to use. This file should be a regular Erlang config file.

The default config file (used by the edis daemon) is installed at /etc/edis/edis.config

Then you can start it by just running:

edis$ sudo service edis start

Now, You can access to the server with any client. For example, the erldis client from the erlang shell:

edis$ make shell
1> {ok,Client} = erldis:connect(localhost,6379).
{ok,<0.81.0>}

Also with the redis client:

$ redis-cli
