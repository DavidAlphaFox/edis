compare/4

compare(Module::atom(), Function::atom(), MathFunction::atom() | fun((pos_integer()) -> number()), Options::[option()]) -> float()

Compares the different runs of Module:Function to a given function. Returns the standard deviation of the distances between them (outliers excluded). The higher the value the more different functions are.
